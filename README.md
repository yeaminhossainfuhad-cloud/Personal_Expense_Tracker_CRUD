# Personal Expense Tracker

A lightweight, native web application constructed using vanilla HTML, CSS, and modern JavaScript to seamlessly execute full **CRUD execution loops** for daily expense tracking metrics.

## Features Delivered

### Core Structural Requirements
- **Complete CRUD Controls**: Add, read logs, dynamically modify values, and drop entries directly out of live tables.
- **Data Validation Safeguards**: Strict validation checks mapping mandatory inputs alongside non-zero/non-negative asset caps.
- **Dynamic Aggregation Analytics**: Live-updating analytic cards displaying cumulative balances, total logs, high/low records, and floating averages.

### Integrated Bonus Modifiers
- **Local Storage Continuity**: State variables map instantly to browser local storage to persist data across page updates.
- **Native CSV Compiler Engine**: Dynamic document construction formatting local items directly into native layout CSV spreadsheets.
- **Live Category Allocation Charts**: Full dashboard distribution tracking mapping visual weight segments via optimized CDN Chart.js components.
- **System Wipe Mechanism**: Universal data deletion button complete with prompt execution confirmation overlays.
